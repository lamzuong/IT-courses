C — Dark with accent (Supabase / Resend)

Dark base with a vibrant signature accent (green here, but could be any color). More "branded", more memorable. Risk: can date faster than monochrome.

My recommendation: A. Quietly polished, lets the content (especially live drag demos) be the star. We can add a subtle accent later if it feels too cold.
