## 05 — Wire the chat endpoint (45 minutes)

For week one, **skip auth and intent classification** — both come in week two. The week-one chat endpoint is the minimum:

```ts
// app/api/chat/route.ts
import { streamText, convertToModelMessages } from 'ai';
import { gateway } from '@ai-sdk/gateway';
import { retrieve } from '@/lib/rag/retrieve';

export async function POST(req: Request) {
  const { messages, locale = 'vi' } = await req.json();

  const lastUser = lastUserText(messages);
  const docs = await retrieve(lastUser, { locale, topK: 5 });

  const context = docs.map((d, i) => `[${i + 1}] (${d.source})\n${d.content}`).join('\n\n---\n\n');
  const system = locale === 'vi'
    ? `Bạn là concierge của shop. Trả lời ngắn gọn từ CONTEXT bên dưới. Nếu CONTEXT không có, nói "em chưa rõ ạ".\n\nCONTEXT:\n${context}`
    : `You are the shop concierge. ...`;

  const result = streamText({
    model: gateway('anthropic/claude-sonnet-4-6'),
    system,
    messages: convertToModelMessages(messages),
  });

  return result.toUIMessageStreamResponse();
}

function lastUserText(messages: any[]): string {
  const u = [...messages].reverse().find(m => m.role === 'user');
  if (!u) return '';
  if (typeof u.content === 'string') return u.content;
  return (u.parts ?? []).find((p: any) => p.type === 'text')?.text ?? '';
}
```

A minimal UI at `app/page.tsx`:

```tsx
'use client';
import { useChat } from '@ai-sdk/react';
import { useState } from 'react';

export default function Home() {
  const [locale, setLocale] = useState<'vi' | 'en'>('vi');
  const { messages, sendMessage, status } = useChat({
    api: '/api/chat',
    body: { locale },
  });

  return (
    <main className="max-w-2xl mx-auto p-6 space-y-4">
      <select value={locale} onChange={e => setLocale(e.target.value as any)}>
        <option value="vi">Tiếng Việt</option>
        <option value="en">English</option>
      </select>
      <div className="space-y-3">
        {messages.map(m => (
          <div key={m.id} className={m.role === 'user' ? 'text-right text-blue-700' : ''}>
            <b>{m.role}:</b>{' '}
            {(m.parts ?? []).map((p, i) => p.type === 'text' ? <span key={i}>{p.text}</span> : null)}
          </div>
        ))}
      </div>
      <form onSubmit={e => {
        e.preventDefault();
        const fd = new FormData(e.currentTarget);
        const text = fd.get('q') as string;
        if (text) sendMessage({ text });
        e.currentTarget.reset();
      }}>
        <input
          name="q"
          autoFocus
          disabled={status === 'streaming'}
          placeholder="Hỏi gì cũng được..."
          className="border w-full p-2 rounded"
        />
      </form>
    </main>
  );
}
```

## 07 — What you have, what's missing

You have:

- A working chat endpoint with hybrid streaming response.
- Postgres + pgvector storing real chunks of your shop's FAQ.
- Bilingual retrieval gated by `locale`.
- An AI Gateway connection that can swap providers in one line.

What's missing — and you don't need this in week one:

- **Authentication.** Anyone can hit `/api/chat` right now. Week two.
- **Tools.** No way to look up live orders or stock. Week two.
- **Write actions.** No way to create an order. Week three.
- **Cost cap and rate limits.** No protection from a runaway loop. Week four.
- **Deployment.** Still on localhost. Week four.

That gap is the *point*. By the end of day five, you've validated the stack at the smallest scale. Each missing piece is now a concrete next step, not a vague mystery.
